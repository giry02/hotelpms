## 5. 실패 및 차단 사항

### CYCLE-ARCH-001 - BLOCKED

Admin과 PMS가 각각 브라우저 `localStorage`를 사용하는 정적 앱 구조다. Admin에서 승인한 호텔·계정·권한이 서버 API를 통해 PMS 테넌트로 전파되는 실제 연동은 현재 구조로 검증할 수 없다.

### CYCLE-E2E-001 - BLOCKED

기존 자동화는 테스트 간 데이터 오염을 막기 위해 업무 흐름별로 새 브라우저 컨텍스트를 사용한다. 따라서 CYCLE-000에서 생성한 동일 호텔·예약·단체·주문·지출·정산 ID를 CYCLE-140까지 이어 쓰는 연결형 실행은 아직 PASS 조건을 충족하지 못했다.
